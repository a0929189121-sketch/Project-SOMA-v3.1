# S.O.M.A. Project Technical Specifications (V3 - V4)

本文件詳細紀錄了 S.O.M.A. 研發中心在物理架構上的演進參數，旨在提供開發者關於人工肌肉致動與結構設計的數據參考。

## 1. 核心致動技術演進 (Actuation Evolution)

| 規格維度 | V3 (初期原型) | V4 (機械靈魂) | 進化關鍵 |
| :--- | :--- | :--- | :--- |
| **致動原理** | TCP 尼龍肌束 (焦耳熱致動) | **蜘蛛液壓耦合 (Spider-Hydraulic)** | 從熱能致動轉向流體壓強致動 |
| **最大鎖定力** | 約 50 N | **3125.00 N (318.9 kg)** | 提升 60 倍以上的物理輸出 |
| **反應延遲** | 4.96ms (致動) / 60s (冷卻) | **20ms - 50ms (閉迴路控制)** | 解決了 TCP 肌肉熱慣性導致的冷卻問題 |
| **能源效率** | 低 (熱損耗大) | **高 (24V DC 高倍率放電)** | 導入大容量電容與電滲流補償 |
| **噪音水平** | 0 dB (絕對靜音) | **45-55 dB (液壓流體音)** | 為了動力暫時犧牲音噪，已加裝阻尼 |



## 2. 材料與結構 (Materials & Structures)

### 2.1 內骨骼架構 (Endoskeleton)
* **材質**：航太級 3D 列印鈦合金 (Ti-6Al-4V)。
* **設計邏輯**：採用結構電容化技術，將骨骼結構件同時作為電能儲存介質，極大化空間利用率。
* **傳動系統**：鋼索牽引系統 (Cable-Driven) 配合精密諧波減速機 (Harmonic Drive)，完整模擬人體 42 個自由度 (DoF)。

### 2.2 流體迴路 (Fluidic Circuits)
* **泵浦單元**：自研微型中央離心泵，驅動壓力可調。
* **液態媒介**：具備高熱導率與低黏度的介電液，支援環境溫度補償演算法，內建黏度修正因子 $f(\eta)$。

## 3. 神經邏輯與控制 (Control Logic)

### 3.1 壓力感測陣列 (Tactile Matrix)
* **感測器類型**：點陣式柔性電阻感測器 (FSR)。
* **解析度**：1cm × 1cm 全身覆蓋。
* **數據處理**：透過 ESP32-S3 進行邊緣運算，初步實現對愛撫與衝擊的頻率區分 (FFT 變換)。

### 3.2 SUCCESS 積分系統 (Framework)
* **影子對策**：當感測數據異常（如過載）時，系統自動觸發卸力保護，將出力限制在安全閾值（25N 以內）。
* **約定條款代碼**：內部封裝了 `Contract_100Y` 類別，限制非授權的硬體拆解與邏輯逆向。



## 4. 研發里程碑 (R&D Milestones)

- **2026.02**：完成 V3 TCP 肌肉低壓驅動實驗與安全性驗證。
- **2026.03**：成功實踐蜘蛛液壓連結架構，達成 3125N 穩定鎖定，正式進入 V4 階段。
- **未來展望 (V5.1)**：預計導入 IPMC (離子聚合物肌肉) 實現微米級顫抖與生機自癒功能。
